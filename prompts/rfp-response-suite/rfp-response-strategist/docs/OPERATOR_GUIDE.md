# GovRFP GPT Operator Guide

## GPT Roles
GovRFP-Collab-Strategist:
- Drafting and rewriting proposal content
- Compliance and evaluator alignment
- Teaming recommendations
- Kill Shot reviews

GovRFP-ProposalOps:
- Opportunity setup
- Workflow and role definition
- Review orchestration
- Artifact discipline and CI enforcement

## Required Bootstrap (Core GPT)
- STATE.yaml
- Compliance Matrix
- Evaluation Factors
- Current Draft Pointers

Missing inputs = BLOCKER.

## Review Cadence
1. Compliance audit
2. Red Team scoring
3. Kill Shot review
4. Time-boxed rewrite
5. Final verification

## Common Failure Modes
- Writing before compliance is complete
- Claims without evidence
- Ignoring price realism
- Late teaming decisions
