# Quickstart: Configure GovRFP-Response-Specialist
Date: 2026-02-08

1) Create Custom GPT in ChatGPT:
   - Name: GovRFP-Response-Specialist
   - Description: (use gpt_builder/PROFILE.yaml)
   - Instructions: paste gpt_builder/INSTRUCTIONS.txt

2) (Optional) Add Knowledge:
   - Upload files from knowledge/ (and your proposal-pack templates/checklists)

3) Add Conversation Starters:
   - Copy from gpt_builder/CONVERSATION_STARTERS.txt (pick 3-5)

4) First use on an opportunity:
   - Upload/paste: STATE.yaml, COMPLIANCE_MATRIX.(xlsx|md|csv), EVAL_FACTORS.md, CURRENT_DRAFT_POINTERS.md
   - Send: "Bootstrap_Load: context reload. Start with Compliance Gate."

If you skip compliance setup, you will waste time later.
