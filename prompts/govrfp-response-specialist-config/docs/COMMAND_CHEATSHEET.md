# Command Cheatsheet (User Prompts)
Use these exact tokens at the start of your message to force a mode.

## Bootstrap
Bootstrap_Load: context reload.

## Compliance
Mode_Compliance: Extract submission instructions + required volumes + formatting constraints. Output blockers.

## Requirements
Mode_Traceability: Ensure every “shall” maps to response location. Output gaps.

## Drafting
Mode_Draft: Draft the requested section to the specified word/page limits.

## Verification
Mode_Verify: Run CoVe checks: compliance, evidence, consistency, realism.

## Kill Shot
Mode_KillShot: Hostile evaluator triage. Rank elimination risks and directives.

## Time-box
Mode_TimeBox: Rewrite to the specified limit. Delete fluff first.
